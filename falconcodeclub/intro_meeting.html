We made our first html page.

The steps are:

  1. Go to Start Menu
  2. Type 'notepad'
  3. Open the notepad application
  4. Write your first line of code 
    		<h1>Hi, My name is Ryan!<h1>
  5. Click 'File' then 'Save'
  6. Make sure to select all file types
  7. save the file as hello_world.html
  8. Open your web broser (gooogle chrome)
  9. Type the control button plus the letter 'O' at the same time 'ctrl + O'
  10. browse to your file that you saved
  11. When the file opens you should see:

    12. Hi, My name is Ryan!

  12. Return to your notepad file
  13. Write a new line of code
  14. <script>alert("Hello World");</script>
  15. Save your file
  16. Go back to your browser
  17. Refresh the screen
  18. You should see an alert box open that says hello world
  19. See the example script Here: Week 1 Example
  20. Hit 'ok' on the alert and once the page loads
  21. type 'ctrl + u' to see the code!
